# `fabl-website`
> the website for FABL, splatoon's first draft major.

![license][license-shield]
![stars][stars-shield]

A website to track the status of FABL.
- Showcase standings from an api
- Show the status of qualifiers

## local setup
1. Clone the repo.
2. `npm i`
3. `npm run dev`
